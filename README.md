# 💻 TaskFlow - Frontend

Frontend de **TaskFlow**, desarrollado con React y Vite.

La aplicación consume la API desarrollada en Express y permite administrar Workspaces mediante una interfaz moderna, responsive e intuitiva.

---

# Tecnologías

- React
- Vite
- React Router
- Context API
- CSS3

---

# Funcionalidades

- Registro de usuarios
- Inicio de sesión
- Verificación de correo
- Recuperación de contraseña
- Crear Workspaces
- Editar Workspaces
- Eliminar Workspaces
- Búsqueda de Workspaces
- Diseño Responsive

---

# Instalación

```bash
npm install
```

```bash
npm run dev
```
